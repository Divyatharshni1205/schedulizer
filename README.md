📅 Schedulizer – Intelligent Timetable Generation System

Schedulizer is an automated timetable generation system designed to efficiently create class and faculty schedules for educational institutions. The system handles complex scheduling constraints such as multiple classes, subject requirements, teacher availability, lab prioritization, blocked slots, and mandatory lunch breaks.

The goal of this project is to eliminate the time-consuming and error-prone process of manual timetable creation by using a structured, constraint-based scheduling approach.

🚀 Features

Generate timetables for 80+ classes simultaneously

Handle multiple subjects per class

Lab hour prioritization for practical sessions

Automatic lunch break allocation between 12 PM – 2 PM

Support for blocked time slots

Prevent teacher schedule conflicts

Generate both:

📚 Class Timetable

👨‍🏫 Teacher Timetable (based on Teacher ID)

Flexible subject configuration including:

Subject Name

Subject Code

Teacher Name

Teacher ID

Theory Hours per Week

Lab Hours per Week

Credits

Blocked Slots

🧠 Problem It Solves

Educational institutions often struggle with manual timetable scheduling due to multiple constraints such as subject hours, teacher availability, and lab requirements. Schedulizer automates this process, ensuring conflict-free and optimized schedules while saving significant administrative time.
